# Shell-Dashboard
Shell-themed Power BI People Analytics &amp; Projections dashboard with interactive Region/Dept slicers, rating correlation, and profit benchmarking by region.

This Power BI report analyzes employee performance and business outcomes using a simple star-style model (Emp/Dept/Region lookups feeding a Performance Review fact table). It includes interactive slicers for Region and Dept Name, an employee detail table (Employee ID, Years of Experience, Dept, Region, Overall Rating Avg), a Boss vs Peer rating correlation scatter, Profit Returned by Region split by Dept Category (Centralized vs Distributed), and a comparison of Overall Rating by Employment Category (Employed vs Fired). The layout uses Shell branding and a dark theme for an executive-style dashboard experience.
